# @sneakylenny/create-inertia-hono

Scaffold a new [Hono](https://hono.dev/) + [Inertia.js v3](https://inertiajs.com/) + [Vue 3](https://vuejs.org/) + [Vite](https://vitejs.dev/) + [Tailwind CSS](https://tailwindcss.com/) project.

## Usage

```bash
npm create @sneakylenny/inertia-hono@latest
# or
bun create @sneakylenny/inertia-hono
# or
pnpm create @sneakylenny/inertia-hono
```

Pass a target directory as the first argument, or run the command without arguments and enter a project name when prompted. The CLI copies the template, sets `package.json` `name` to the folder name, then suggests `install` and `dev` using your detected package manager.
